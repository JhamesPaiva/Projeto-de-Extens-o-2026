# Projeto-de-Extens-o-2026
Plataforma web para divulgação de eventos comunitários, conectando organizadores e participantes de forma simples e acessível. Permite listar, buscar e visualizar eventos por categoria, data e local, promovendo engajamento e fortalecimento da comunidade local.

# 🌐 Portal de Eventos Comunitários

## 📌 Sobre o projeto

O **Portal de Eventos Comunitários** é uma plataforma web desenvolvida com o objetivo de divulgar eventos locais, facilitando o acesso da comunidade a atividades culturais, sociais, esportivas e educacionais.

A proposta é conectar organizadores e participantes de forma simples, rápida e acessível.

---

## 🚀 Funcionalidades

* 📅 Listagem de eventos
* 🔎 Filtro por categoria, data e local
* 📄 Visualização detalhada do evento
* 📝 Cadastro de novos eventos
* 📱 Layout responsivo (mobile e desktop)
